# Adobe[!DNL Target] Standard和Premium的訓練影片

觀看訓練影片來協助您善用 Target Standard 和 Target Premium。

影片分成下列類別:

* Experience Cloud 和一般功能
* 建立活動
* 對象和鎖定目標
* 資產管理
* 設定和偏好設定
* 實施和疑難排解
* 行銷使用案例
