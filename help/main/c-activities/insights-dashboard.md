# Adobe Target見解控制面板

[!UICONTROL Adobe Target Dashboard]提供貴組織如何隨著時間使用Target的高階檢視。 它有助於團隊一眼就瞭解採用、活動量和實驗用法。

此頁面專為希望快速瞭解Target使用情形而無需分析個別報告的從業人員與利害關係人設計。

## 標頭圖說文字：Experimentation Accelerator

在頁面頂端，儀表板包含參考&#x200B;[!UICONTROL **Experimentation Accelerator**]&#x200B;的簡短標題訊息。 此圖說文字旨在讓客戶瞭解有助於簡化實驗工作流程的其他功能。 對於想要探索旨在簡化實驗設定、分析和決策的工具的客戶，此訊息是一個輕量型的入口點。

## 時間範圍選取

在頁面頂端，您可以選取時間範圍（例如，上週、去年或所有時間）。

選取的時間範圍會一致地套用至頁面上的所有度量和圖表。

**重要須知：**

* 有些量度會反映此時間範圍內任何時間點上線的活動。
* 其他則反映在該時間範圍內建立、發佈或完成的活動。
* 因此，各量度的總計不一定都會完全相加。 例如，許多活動可能都在相同的時間範圍內開始和完成。

## 摘要量度

頁面頂端的摘要測量結果，可提供所選時間範圍內活動的快速快照。

### 已上線活動總數

此量度會顯示所選時間範圍內任何時間點上線的活動數量。

如果活動主動提供流量，即使活動在所選期間之前或之後開始，仍會視為已上線。

**使用此量度：**

* 瞭解在此期間使用Target的積極性
* 評估個人化和測試工作的整體規模

### 活動已上線或已修改

貴組織在所選時間範圍內已上線、建立或修改的活動總數。

**使用此量度：**

* 瞭解Target活動程式庫的整體大小以及正在使用的活動數目
* 追蹤實驗和個人化方案的長期成長

### 已完成的活動

此量度會顯示所選時間範圍內達到完成或停止日期的活動數量。

**使用此量度：**

* 瞭解在該時段內完成的活動數
* 追蹤一段時間的完成量

### 已發佈活動

此量度會顯示所選時間範圍內已發佈的活動數量。

活動首次上線時會被視為已發佈。 如果活動上線、停止，然後再次上線，此量度中只會計算第一次發生次數。

**使用此量度：**

* 測量已啟動的新活動數
* 瞭解活動建立和發佈速度

## 活動型別劃分

「活動型別」圖表會依型別顯示活動的分佈，例如：

* A/B 測試
* 體驗鎖定 (XT)
* Automated Personalization (AP)
* 多變數測試 (MVT)

此圖表會反映所選時間範圍內的活動總數。

**使用此區段來：**

* 瞭解最常使用的Target功能

## A/B測試量度

本節著重說明與A/B測試相關的使用方式。

### 即時A/B測試活動總數

顯示所選時間範圍內任何時間點上線的A/B測試活動數目。

### 已發佈的A/B測試總數

顯示所選時間範圍內已發佈的A/B測試活動數。

**使用這些量度：**

* 瞭解A/B測試的使用頻率
* 追蹤實驗數量和採用率

## 解譯資料

檢閱此控制面板時，請記住下列事項：

* 量度可能包含在所選時間範圍之前開始或結束的活動。
* 根據其生命週期（例如，已發佈及已完成），活動可能會計入多個量度中。
* 此儀表板專注於使用和採用，而不是績效結果。

如需詳細的結果、提升度或統計效能，請參閱Adobe Target中的個別活動報表。

## 預期用途

此儀表板非常適合用於監控Target採用和使用趨勢。 它並非旨在取代詳細的活動層級分析。
