# 活動變更記錄

[!DNL Adobe Target] [!UICONTROL 變更記錄檔]會記錄活動動作，包括變更的內容、變更者及變更發生的時間。 已填入[!UICONTROL 變更記錄]頁面以進行即時分析。

若要存取[!UICONTROL 變更記錄檔]頁面：

1. 按一下「**[!UICONTROL 活動]**」>從清單中按一下所需的活動。
1. 按一下左側邊欄中的&#x200B;**[!UICONTROL 變更記錄檔]**。

每次建立、更新、啟動或停用活動時，就會列出執行的動作、執行動作的日期以及執行動作的使用者。 活動中使用的自訂定位與選件之變更也會列出。

重新整理頁面以查看狀態變更。 您也可以按一下[!UICONTROL Date]欄標題，以遞增或遞減順序列出執行的動作。
