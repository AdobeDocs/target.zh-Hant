# 控制多個特徵的特徵群組 {#feature-groups}

[功能標幟](what-is-a-feature-flag.md)控制單一功能。 當您需要一起管理多個相關功能標幟，並確保它們觸及相同的對象時，您使用&#x200B;**功能群組**。

## 功能群組的功能 {#what-it-does}

功能群組隨附多個功能標幟，並可讓您將單一對象規則一次套用至所有標幟。 這在單一使用者面對功能跨越多個應用程式或服務時相當實用。

例如，假設共同作業功能涉及整個案頭應用程式、行動應用程式和後端服務的變更。 透過使用所有三個應用程式的旗標建立功能群組，您可以對1%的使用者開啟功能，並確保這些使用者同時在所有三個表面上看到一致的體驗。

## 跨應用程式分組 {#cross-application}

功能群組支援跨應用程式的功能管理。 可將跨多個應用程式的相關標幟分組。
