# 在對象規則中使用內容 {#context-in-audience-rules}

內容變數是使用者端應用程式在執行階段提供的值。 它們可讓您根據動態的工作階段層級資訊來鎖定使用者，例如使用者的作用中語言、裝置型別或應用程式狀態。

上下文變數與Web和行動使用者端相關。

## 上下文變數的運作方式 {#how-context-works}

評估功能標幟時，您的應用程式會將內容變數傳遞給「標幟」。 您可以在主控台中定義規則來檢查這些值，平台會在評估時使用它們來判斷使用者是否符合條件。

## 內容變數型別 {#variable-types}

### 預先定義（清單）型別 {#predefined-type}

上下文變數，具有一組固定的允許值，例如語言或國家/地區清單。

可用的運運算元： **是**，**不等於**，**存在**，**在**

### 整數型別 {#integer-type}

儲存數值的內容變數。

可用的運運算元： **大於**、**大於或等於**、**是**、**小於**、**小於或等於**

### 字串型別 {#string-type}

儲存自由格式文字值的內容變數。

可用的運運算元： **Is**，**不等於**

## 新增內容變數 {#adding-context-variable}

若要將內容變數新增至對象規則：

1. 在主控台中開啟功能標幟或功能群組。
2. 前往&#x200B;**對象**&#x200B;標籤。
3. 在&#x200B;**內容**&#x200B;下，新增條件。
4. 選取內容變數、運運算元和值。

如果您所需的內容變數未出現在清單中，您可以從主控台的內容變數管理區段以自助方式建立新的內容變數。
