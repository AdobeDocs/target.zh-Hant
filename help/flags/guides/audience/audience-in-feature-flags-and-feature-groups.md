# 功能標幟和功能群組中的對象 {#audience-overview}

標幟為功能標幟和功能群組提供受眾目標定位。 您可以結合上下文變數，以精確控制哪些使用者會收到功能。

如需以內容為基礎的鎖定目標（例如以使用者的使用中語言或裝置型別鎖定目標），請參閱[在對象規則中使用內容](using-context-in-audience-rules.md)。
