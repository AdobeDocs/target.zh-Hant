# Analytics {#analytics}

Flags為功能標幟、功能群組、跨團隊功能群組及發行提供內建分析。 使用Analytics儀表板來瞭解有多少使用者參與您的轉出，以及變體和控制組如何比較。 您也可以將標幟資料匯出至您偏好的報表環境，以便與其他Adobe資料一起進行分析。

## 啟用分析 {#enable}

Analytics必須在兩個層級啟用：

1. **應用程式層級** — 連絡人旗標支援可為您的應用程式啟用分析。
2. **功能標幟層級** — 為您的應用程式啟用分析後，請勾選您要追蹤之每個功能標幟的&#x200B;**基本詳細資料**&#x200B;索引標籤上的&#x200B;**啟用分析**&#x200B;核取方塊。

>[!NOTE]
>
>依預設，每個應用程式最多可啟用20個功能標幟。 如果您需要提高此限制，請聯絡支援人員。

## 檢視Analytics控制面板 {#dashboard}

啟用分析後，應用程式的所有功能標幟、功能群組和發行版本都將開始追蹤資料。 在您要分析的功能標幟、功能群組或版本上選取&#x200B;**結果**，即可存取儀表板。

控制面板會顯示：

* **參與者** — 符合功能資格的使用者總數（變體+控制組組合）
* **控制組** — 指派給控制組的使用者數目（收到預設體驗的使用者）
* **日層級圖表** — 顯示一段時間內變體和控制組註冊情況的每日折線圖；標籤表示功能旗標組態何時更新
* **變體層級分析** — 已註冊控制組和每個變體的使用者累計計數

對於功能群組和版本，請選取&#x200B;**結果**&#x200B;下拉式清單以挑選應用程式並檢視該應用程式的分析。 Analytics僅適用於啟用它的應用程式。
