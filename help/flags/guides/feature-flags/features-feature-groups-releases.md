# 特徵和特徵群組 {#features-feature-groups}

標幟提供兩種用於管理功能轉出的成品。 選擇正確的轉出取決於您的轉出範圍以及所涉及的特徵數量。

## 兩個成品 {#artifacts}

**功能標幟**
最原子化的單位。 控制單一應用程式的單一功能。 可以為定義的對象啟用或停用。

**功能群組**
屬於相同團隊的功能旗標集合。 允許以單一單元管理多個應用程式上的多個標幟。

## 比較 {#comparison}

| | 功能標幟 | 功能群組 |
|---|---|---|
| **管理對象所需的角色** | 產品發行擁有者 | 產品發行擁有者 |
| **應用程式** | 單個 | 多個（相同團隊） |
| **豐富對象條件** | ✓ | ✓ |
| **轉出百分比** | 與任何對象條件結合 | 與任何對象條件結合 |
| **A/B測試** | ✓ | ✓ |
| **自助服務** | ✓ | ✓ |
| **稽核軌跡** | ✓ | ✓ |

## 何時使用每個 {#when-to-use}

| 藍本 | 使用 |
|---|---|
| 針對一個應用程式測試或推出單一功能 | **功能標幟** |
| 在同一個團隊中協調多個功能，同時上線 | **功能群組** |
