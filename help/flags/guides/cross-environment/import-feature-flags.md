# 匯入功能標幟 {#import-feature-flags}

旗標可讓您將功能旗標從一個沙箱（例如，沙箱1）匯入到另一個沙箱（例如，沙箱2）中。 這可避免手動重新建立標幟設定，並降低沙箱之間設定漂移的風險。

## 步驟1：前往目的地沙箱和應用程式 {#step-1}

登入&#x200B;**目的地**&#x200B;沙箱的主控台 — 您要將旗標&#x200B;*匯入*&#x200B;的沙箱。 從「功能旗標」頁面的應用程式下拉式清單中，選取您要匯入旗標的應用程式。

>[!IMPORTANT]
>
>您目前的沙箱和選取的應用程式必須是&#x200B;**目的地** — 而不是來源。 例如，若要將旗標從沙箱1匯入沙箱2，請登入沙箱2主控台，並選取您的沙箱2應用程式。

## 步驟2：開啟匯入對話方塊 {#step-2}

選取&#x200B;**匯入功能旗標**。 隨即開啟對話方塊，根據可用的應用程式預先填入來源沙箱和應用程式。 如有需要，您可以從對話方塊的下拉式清單中變更來源沙箱和應用程式。

## 步驟3：選取要匯入的功能標幟 {#step-3}

從來源沙箱中的功能標幟清單中，選取您要匯入的標幟。 您可以一次選取一個、多個或所有標幟。

## 步驟4：選取要匯入的功能旗標狀態 {#step-4}

使用下拉式清單來選擇應如何匯入功能標幟 — **已啟用**、**已停用**，或處於其&#x200B;**目前狀態**。 依預設，功能標幟會以&#x200B;**已停用**&#x200B;狀態匯入。

## 重要附註 {#important-notes}

匯入功能標幟時，請記住下列事項：

* 如果目的地沙箱中已存在具有相同索引鍵的功能標幟，則不會匯入該功能。
