# 作業系統 {#operating-system}

您可以鎖定使用某個作業系統的訪客。

1. 在 [!DNL Target] 介面中，按一下**[!UICONTROL 「對象]** &gt; **[!UICONTROL 建立對象」]**。
1. 為對象命名。
1. 按一下**[!UICONTROL 「新增規則]** &gt; **[!UICONTROL 作業系統」]**。

   ![](assets/target_os.png)

1. 按一下**[!UICONTROL 「選取」]**，然後選取下列其中一個選項:

   * Linux
   * Macintosh
   * Windows

1. (可選) 按一下**[!UICONTROL 「新增規則」]並設定對象的其他規則。**
1. 按一下**[!UICONTROL 「儲存」]**。
